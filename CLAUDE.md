# Claude Integration Guide  

## Purpose  
Document guidelines and prompts optimized for Anthropic Claude models (e.g., Claude 3.5 Sonnet/Opus).  

## Prompting Patterns  
- Use constitution-style long-form hints for high-level tasks.  
- Define clear role structure and tool invocation expectations.  
- Use map-reduce patterns to handle long contexts.  
- Provide JSON-based output formats.  
- Include parameters for tool use mode.  

## Cheat Sheet  
Include a quick reference for best prompting patterns, an evaluation harness for comparing outputs, and example system prompts for common development tasks.  

## Parameters  
Note recommended parameters (model versions, temperature, max tokens) and tool-use API usage guidelines.
